# Marketing assets — folder index

All launch marketing files live under `public/`. **Do not rename top-level folders** without updating deployment and documentation.

| Folder | Purpose |
|---|---|
| [`videos/`](./videos/) | Homepage cinematic intro + poster |
| [`screenshots/`](./screenshots/) | Web, iOS, Android store screenshots |
| [`og/`](./og/) | Open Graph and social share images |
| [`press/`](./press/) | Press kit PDFs, fact sheets, media copies |
| [`logos/`](./logos/) | Logo SVG/PNG exports and favicon sources |
| [`app-preview/`](./app-preview/) | App Store & Play Store preview videos |
